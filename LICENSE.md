## Responsible Use

This software operates with elevated system privileges, directly manipulates
kernel structures, and modifies Windows Image File Execution Options registry
entries. It is intended for:

- **Authorized security testing and penetration testing**
- **Security research and education**
- **System administration in controlled environments**
- **Malware analysis and incident response**
